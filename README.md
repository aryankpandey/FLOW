🎧 FocusSpace — Ambient Productivity Sanctuary

«Enter deep work. Stay focused. Build momentum.»

FocusSpace is a minimalist productivity dashboard designed to help users achieve a state of deep focus through structured work sessions, immersive ambient audio, and intentional daily planning.

By combining a customizable Pomodoro timer, layered ambient soundscapes, and a lightweight intention tracker, FocusSpace creates a distraction-free environment where productivity feels natural and sustainable.

✨ Features

⏳ Customizable Pomodoro Canvas

Stay productive using a smooth circular countdown timer inspired by the Pomodoro Technique.

- Adjustable work sessions
- Short break intervals
- Long break intervals
- Visual progress indicator
- Session completion notifications

---

🎵 Ambient Soundboard Mixer

Create your ideal focus environment by mixing multiple ambient sounds simultaneously.

Available soundscapes:

- 🌧️ Rain
- ☕ Cafe Ambience
- 🎼 Lo-Fi Music

Features:

- Independent volume controls
- Real-time audio mixing
- Layer multiple tracks together
- Persistent playback controls

---

📝 Daily Intentions Grid

Plan your day with a clean and distraction-free intentions board.

Features:

- Create daily focus goals
- Mark tasks as completed
- Delete tasks instantly
- Data persistence via Local Storage
- Bento-grid inspired interface

---

💡 Zen Quotes Integration

Receive motivational and reflective quotes throughout your focus sessions.

Features:

- Dynamic quote fetching
- Inspirational content updates
- Lightweight API integration

Powered by the ZenQuotes API.

🎯 How It Works

1. Set your preferred Pomodoro durations.
2. Start a focus session.
3. Mix ambient sounds to create your ideal workspace.
4. Add daily intentions and tasks.
5. Stay motivated with rotating Zen quotes.
6. Complete your sessions and maintain momentum.
